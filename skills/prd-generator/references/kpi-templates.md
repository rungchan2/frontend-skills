# 도메인별 KPI 템플릿

## SaaS / B2B

### 핵심 지표
| 지표명 | 설명 | 측정 방법 | 목표 예시 |
|--------|------|----------|----------|
| MRR (Monthly Recurring Revenue) | 월간 반복 매출 | 구독료 × 활성 구독자 수 | $10K/월 |
| ARR (Annual Recurring Revenue) | 연간 반복 매출 | MRR × 12 | $120K/년 |
| Churn Rate | 이탈률 | 이탈 고객 / 전체 고객 × 100 | < 5%/월 |
| CAC (Customer Acquisition Cost) | 고객 획득 비용 | 마케팅 비용 / 신규 고객 수 | < $100 |
| LTV (Lifetime Value) | 고객 생애 가치 | ARPU / Churn Rate | > $500 |
| LTV:CAC Ratio | 투자 효율성 | LTV / CAC | > 3:1 |

### 제품 지표
| 지표명 | 설명 | 측정 방법 | 목표 예시 |
|--------|------|----------|----------|
| DAU/MAU Ratio | 사용자 참여도 | 일간 활성 / 월간 활성 | > 20% |
| Feature Adoption Rate | 기능 채택률 | 기능 사용자 / 전체 사용자 | > 30% |
| Time to Value | 가치 실현 시간 | 가입 → 첫 성공 액션 | < 5분 |
| NPS (Net Promoter Score) | 추천 의향 | 추천자 - 비추천자 | > 50 |
