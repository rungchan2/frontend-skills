## 행동 패턴별 페르소나

### 얼리어답터
- **특징**: 새로운 기술/서비스 적극 시도
- **동기**: 혁신, 남들보다 앞서기
- **채널**: 테크 블로그, 트위터, 레딧
- **가격 민감도**: 낮음 (가치 중시)
- **피드백**: 적극적, 상세함

### 실용주의자
- **특징**: 검증된 솔루션 선호
- **동기**: 효율성, 문제 해결
- **채널**: 리뷰, 추천, 지인 소개
- **가격 민감도**: 중간 (가성비 중시)
- **피드백**: 필요시에만

### 보수적 사용자
- **특징**: 변화에 저항, 익숙한 것 선호
- **동기**: 안정성, 신뢰
- **채널**: 전통 미디어, 입소문
- **가격 민감도**: 높음
- **피드백**: 거의 없음
