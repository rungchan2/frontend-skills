## 경로 표현

화살표(→)로 네비게이션 경로 표현:

```
**가맹점 대시보드** → **원비청구** → **청구서 생성**
```

**Notion rich_text:**
```json
[
  { "type": "text", "text": { "content": "가맹점 대시보드" }, "annotations": { "bold": true } },
  { "type": "text", "text": { "content": " → " } },
  { "type": "text", "text": { "content": "원비청구" }, "annotations": { "bold": true } }
]
```

## 상태 흐름

이모지+색상으로 상태 변화 표현:

```
미납 🔴 → 부분납 🟡 → 완납 🟢
```

```
대기중 ⏳ → 처리중 🔄 → 완료 ✅
```

```
예약 📅 → 확정 ✅ → 완료 🏁
```

## 버튼/메뉴 강조

작은따옴표 + 굵게 + 색상:

```
**'저장'** 버튼을 클릭합니다.
```

**Notion rich_text (색상 포함):**
```json
[
  { "type": "text", "text": { "content": "'저장'" }, "annotations": { "bold": true, "color": "blue" } },
  { "type": "text", "text": { "content": " 버튼을 클릭합니다." } }
]
```

## 입력값 표현

인라인 코드로 형식/예시 표현:

```
전화번호: `010-1234-5678` 형식으로 입력
날짜: `2024-01-15` 형식으로 입력
금액: `300,000` (숫자만 입력)
```

**Notion rich_text:**
```json
[
  { "type": "text", "text": { "content": "전화번호: " } },
  { "type": "text", "text": { "content": "010-1234-5678" }, "annotations": { "code": true } },
  { "type": "text", "text": { "content": " 형식으로 입력" } }
]
```

## 비교 테이블

기능/옵션 비교 시 사용:

| 구분 | 일괄 생성 | 개별 생성 |
|------|-----------|-----------|
| 대상 | 전체 학생 | 특정 학생 |
| 속도 | ⚡ 빠름 | 보통 |
| 용도 | 월초 정기 청구 | 추가/수정 청구 |
| 추천 | ✅ 기본 사용 | 필요 시 |

## 상태 테이블

상태값 설명 시 이모지+색상 필수:

| 상태 | 색상 | 설명 |
|------|------|------|
| ✅ 완납 | 🟢 초록 | 전액 납부 완료 |
| ⏳ 미납 | 🔴 빨강 | 납부 대기 |
| 🔄 부분납 | 🟡 노랑 | 일부 금액만 납부 |
| ❌ 취소 | ⚫ 회색 | 청구 취소됨 |

| 상태 | 색상 | 설명 |
|------|------|------|
| 📅 예약 | 🔵 파랑 | 상담 예약됨 |
| ✅ 확정 | 🟢 초록 | 학부모 확정 |
| 🏁 완료 | ⚫ 회색 | 상담 완료 |
| ❌ 취소 | 🔴 빨강 | 상담 취소 |

| 상태 | 색상 | 설명 |
|------|------|------|
| ⏰ 출근 | 🟢 초록 | 정상 출근 |
| ⏰ 지각 | 🟡 노랑 | 지각 출근 |
| 🏠 퇴근 | 🔵 파랑 | 퇴근 완료 |
| ❌ 결근 | 🔴 빨강 | 미출근 |

## Step 구조 패턴

각 Step은 이 순서로 작성:

```
### Step X. [작업 제목]

[작업 설명 1-2문장]

[이미지 블록 - caption에 스크린샷 설명]

1. **'버튼명'**을 클릭합니다.
2. [다음 동작]
3. [결과 확인]

💡 Tip: [유용한 정보]

---
```

## FAQ 패턴

실제 사용자 관점 질문:

```
> Q: [문제 상황]?
A: [해결 방법]
```

예시:
```
> Q: 청구서를 잘못 생성했어요. 어떻게 해야 하나요?
A: 청구서 상세에서 **'삭제'** 버튼을 클릭하면 삭제할 수 있습니다.

> Q: 학생이 반을 변경했는데 수강료가 안 바뀌어요.
A: 반 변경 후에는 청구서를 **다시 생성**해야 변경된 수강료가 적용됩니다.

> Q: 알림톡은 언제 발송되나요?
A: **'알림톡 발송'** 버튼 클릭 시 즉시 학부모에게 발송됩니다.
```
